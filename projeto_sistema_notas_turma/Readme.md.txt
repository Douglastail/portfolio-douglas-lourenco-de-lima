Sistema de Gestão de Notas 📊
Este projeto consiste em um script Python que automatiza o cálculo de médias e a classificação de alunos de uma turma. Desenvolvido originalmente no Google Colab, o código foca em estruturas de repetição e lógica condicional.

🚀 Funcionalidades
Entrada de Dados Personalizada: O usuário define a quantidade total de alunos antes de iniciar o processamento.

Cálculo Automático: Processa a média aritmética entre duas avaliações.

Feedback de Status:

Aprovado: Para médias iguais ou superiores a 7.0.

Recuperação: Para médias entre 5.0 e 6.9.

Reprovado: Para médias inferiores a 5.0.

🛠️ Tecnologias e Ferramentas
Linguagem: Python 3.

Ambiente de Desenvolvimento: Google Colab.

Versionamento: Git e GitHub.

📂 Como utilizar
Certifique-se de ter o Python instalado.

Baixe o arquivo projeto_sistema_notas_turma.py.

Execute o script no terminal:

Python

python projeto_sistema_notas_turma.py
🧠 Lógica de Programação
O projeto utiliza um loop for para iterar conforme o número de alunos e uma estrutura if-elif-else para a tomada de decisão baseada nos valores de entrada. Este script faz parte de uma série de exercícios de lógica e desenvolvimento que venho organizando em meus repositórios.
