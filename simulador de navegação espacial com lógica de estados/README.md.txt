# 🎨 Processamento de Imagens e Estruturas de Matrizes

## 📝 Descrição do Projeto
Projeto focado na manipulação de baixo nível de dados visuais e sonoros. Através da manipulação de matrizes de pixels (RGB), o sistema aplica filtros de sombreamento em artes digitais e realiza a transposição de dicionários musicais complexos. É um estudo fundamental para as disciplinas de **Interface Humano-Computador (IHC)** e Álgebra Linear.

## 🚀 Tecnologias Utilizadas
* **Linguagem:** Python 3.10
* **Bibliotecas:** Matplotlib (Renderização) e NumPy (Processamento numérico).
* **Conceitos:** Manipulação de canais RGB e transposição de matrizes.

## 📊 Resultados e Aprendizados
O projeto demonstrou como a matemática discreta impacta a computação gráfica.
* **Renderização:** Utilizei `imshow` para transformar arrays numéricos em representações visuais (Pixel Art).
* **Eficiência:** Aprendi a aplicar transformações em lote (batch) percorrendo múltiplos níveis de dicionários e listas.
* **Lógica Digital:** Pratiquei a alteração de luminosidade através da manipulação dos valores decimais de cada canal de cor.

## 🔧 Como Executar
1. Clone o repositório.
2. Instale as dependências: `pip install matplotlib numpy`.
3. Execute o comando: `python pixel_art.py`.
