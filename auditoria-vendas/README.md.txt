# 🛡️ Sistema de Segurança e Auditoria
Este projeto foca na integridade de dados financeiros e monitoramento de vendas.

### 🛠️ Tecnologias e Conceitos
- **Linguagem:** Python[cite: 1].
- **Tratamento de Erros:** Uso de `try-except` para evitar falhas por entradas inválidas de usuários[cite: 1].
- **Lógica de Auditoria:** Cálculo de médias e verificação de discrepâncias (vendas 5x maiores que a média)[cite: 1].
- **Controle de Segurança:** Implementação de variável global `LIMITE_SEGURANCA` para controle de quarentena do sistema[cite: 1].
